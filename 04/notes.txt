** diff between package and module **
module: node js core mai jo installed aata hai use bolte h module.
package: jo hum npm se download krte hai use bolte h package.

eg: agar hume esa app bnana hai jo cheezo ko pdh kr voice mai convert krta hai, to esa app bnane k liye hume "text to speech"
apply krna pdega jisme hume bhaut time lg jayega lekin google ne use pehele hi kr dia h or hum vo package npm store se download
krke use kr skte hai thats it.

** installation and uninstallation of anything basics & advance **
install: comand-> npm install package_name
uninstall: comand-> npm uninstall package_name
install any specific version-> npm install package_name@version
